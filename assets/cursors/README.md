# Cursores

Este tema incluye el pack **W11 Tail Cursor Concept Free** por [Jepri Creations](https://www.deviantart.com/jepricreations).

## Estructura

```
w11-tail-cursor-concept-free/
    cursor/
      dark/   ← usado con NASA_Tema_Oscuro
      light/  ← usado con NASA_Tema_Claro
```

El script de instalación copia automáticamente los cursores correctos según el tema elegido.
